# Touchy Subjects LLC

Static concept gallery for Touchy Subjects visual directions.

## Pages

- `index.html` — gallery launcher for all options
- `oldindex.html` — archive editorial option
- `testpage1.html` — cursor fingerprint option
- `testpage2.html` — fingerprint field option
- `testpage3.html` — chroma heading study
- `testpage4.html` — morphing background option
- `testpageshaders.html` — shader sketchbook option

## Local Preview

Run a simple static server from the repo root:

```bash
python3 -m http.server 4173
```

Then open:

- `http://127.0.0.1:4173/index.html`

## Notes

- These pages are plain HTML/CSS/JS experiments intended for side-by-side review.
- `index.html` is the recommended starting point because it links to every option.
