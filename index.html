Colorful fingerprint textures with live motion

Fingerprints as atmosphere.

A full-bleed texture field built around Purple #8B5CF6, Blue #3B82F6, Green #10B981, Orange #F59E0B, and Red #EF4444. The ridges drift with the pointer, brighten by palette selection, and stay lightweight enough to run as a plain static page.

Interaction

Five texture voices.

The page uses one fingerprint language, then lets each color carry a different mood. The rows below are rendered from SVG ridge paths so the same texture system can be reused outside the hero canvas.

01 / Purple

Signal Bloom

Use this as the anchor tone when you want the fingerprint effect to feel dreamlike rather than clinical. It carries the main glow without flattening the page.

02 / Blue

Cool Drift

Blue adds depth and space. It is the cleanest stroke in the set, so it works well for calmer motion layers and larger background shapes.

03 / Green

Fresh Contour

Green acts like a live edge. It breaks up the cooler palette and keeps the fingerprint clusters feeling organic instead of purely synthetic.

04 / Orange

Pressure Heat

Orange gives the field a tactile warmth. It is strongest in shorter bursts, especially on hover states and edge highlights.

05 / Red

Hotline Ridge

Red is the final high-energy layer. It reads best as a punctuation color, giving the whole system a stronger pulse without overpowering the other tones.

One texture system, three effects.

The page is not leaning on generic glow blobs. It uses nested ridge geometry and then changes where that geometry lives: full-canvas motion in the hero, SVG marks inside sections, and color focus changes from the palette rail.

Entrance

GSAP fades the interface in as a poster, not a dashboard.

Depth

The hero canvas tracks pointer position with slow parallax and phase drift.

Reuse

The same ridge math is reused to generate scalable inline SVG texture marks.

Inline SVG Texture Engine

Static file. Live surface.

This version is deliberately lightweight: one `index.html`, no build step, and only a progressive GSAP enhancement on top. You can drop it onto a static host and still keep the animated fingerprint field and SVG texture system intact.
